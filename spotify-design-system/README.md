# Welcome to 05 Styling
***

## Task
Build a Spotify-inspired design system with utility-first CSS, component styling, dark/light theme switching, and interactive music player UI.

## Description
I built a complete Spotify design system with:
- Tailwind CSS with custom Spotify color tokens
- Dark/Light mode theme switching
- Interactive music player with progress bar and volume control
- Responsive grid layouts for song cards
- Sidebar navigation like Spotify
- Artist dashboard with stats and top tracks
- Search functionality for filtering songs

## Installation
```bash
cd spotify-design-system
npm install --legacy-peer-deps
npm run dev
```

## Usage
- Home: http://localhost:3000
- Dashboard: http://localhost:3000/dashboard
